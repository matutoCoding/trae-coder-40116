# 剧本杀组局桌面客户端

基于 Electron + React + TypeScript + Ant Design 开发的剧本杀组局管理桌面应用。

## ✨ 功能特性

### 🎯 四大核心模块

1. **双向撮合模块**
   - 拼场局登记：创建新局，设置剧本、时间、房间、DM、发起人
   - 双向意愿登记：玩家表达想玩意向，局方发出缺人邀请
   - 互选成交判定：**只有双方都同意才能撮合成功**，单方意愿不成交
   - 自动凑人：人数达标自动成团

2. **契合排序模块**
   - 多维度契合度评分（题材50% + 难度30% + 标签加成20%）
   - 按口味匹配度从高到低排序推荐
   - 支持「按局找玩家」和「按玩家找局」双向模式

3. **优惠计算模块**
   - 支持折扣券（打折，可设上限）和满减券
   - **优惠计算顺序可配置**（先满减后折扣 或 先折扣后满减）
   - 可叠加券和独占券
   - 叠加后金额不为负（最低0元保护）
   - 自动计算最优优惠组合

4. **账单生成模块**
   - 批量为整局玩家生成账单
   - 优惠明细展示（应用顺序、类型、金额）
   - 账单打印
   - 支付状态管理（待支付/已支付/已退款）
   - 营收统计

### 📅 附加功能

- **DM主持排班**：日历视图、空闲/已安排/休息三态
- **玩家管理**：新增、偏好设置
- **剧本管理**：题材、难度、人数、时长、价格
- **数据概览**：运营数据看板

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式（桌面客户端）

**推荐使用此命令进行开发，会自动打开 Electron 桌面应用：

```bash
npm run dev
```

### 开发模式（仅 Web 预览）

如果只想在浏览器中预览 Web 版本：

```bash
npm run dev:web
```

然后访问 `http://localhost:5173`

### 类型检查

```bash
npm run typecheck
```

### 构建打包

#### 打包桌面应用（Windows/macOS/Linux）

```bash
npm run build
```

#### 仅构建 Web 版本

```bash
npm run build:web
```

## 📝 业务规则说明

### 双向撮合规则
- 玩家点击「我想玩」 + 局方点击「局方缺人」 → 双方都为是 → 撮合成功
- 撮合成功的玩家自动加入对局，已加入的玩家不再出现在推荐列表
- 人数达到要求后，局状态变为「已成团」

### 优惠计算规则
- 优惠顺序影响最终价格：
  - 例：原价 ¥200，满100减20，8折券
  - 先满减后折扣：(200-20)×0.8 = ¥144
  - 先折扣后满减：200×0.8-20 = ¥140
- 独占券不可与其他券叠加，系统自动计算最优方案
